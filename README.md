# Mood-Based Music / Quote Recommender

A simple web app that recommends **songs and motivational quotes** based on your mood.  
Built using **Python (Flask)**, **HTML/CSS**, and **JavaScript**.

## Features
- User selects or types their mood
- App suggests relevant songs
- App suggests motivational quotes
- Simple, interactive UI

## Tech Stack
- Python
- Flask
- HTML / CSS / JS
